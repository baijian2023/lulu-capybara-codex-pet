# 水豚噜噜 · Codex Pet

一只橙黄色、头顶小橘子、会比心和转圈圈的 Codex v2 小宠物。

## 下载与安装

下载仓库中的 `水豚噜噜-Codex宠物安装包.zip` 并解压，将 `lulu-capybara` 文件夹复制到：

- macOS / Linux：`~/.codex/pets/`
- Windows：`%USERPROFILE%\.codex\pets\`

重新打开 Codex，在宠物设置中选择“水豚噜噜”。
